# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in NMHub, please open a private issue on GitHub or contact the repository owner directly.

## Scope

- The NMHub loader and modules execute Lua code fetched from GitHub.
- Do NOT share your GitHub tokens or executor credentials through the script.
- The script does not collect or transmit any user data.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
